#  EZLabs Contact Page (React + Tailwind)

A **Single Page Application (SPA)** built using **React**, **Tailwind CSS**, **React Router**, and **Lucide React Icons**.  

##  Prerequisites

- **Node.js** v18 or above  
- **npm** v8 or above
